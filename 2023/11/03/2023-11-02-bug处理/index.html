2023-11-02 bug处理

发表于 2023-11-03 更新于 2023-12-19 分类于 bug 处理 Valine：
本文字数： 752 阅读时长 ≈ 1 分钟

bug描述

前端展示组织树，一直点下一级，点到最后一级时，选择一个组织，同级组织全部被选中。

bug分析

发现返回的组织树的id是长整型。而前端用的是number类型接收。number类型最大范围为9007199254740992。 导致精度丢失，同一级别节点的id变成一样。导致都被选中

解决方案

将long型数字替换成字符串。
后端处理在字段上加 @JsonSerialize(using = ToStringSerializer.class)

引发的思考 前端传String类型可以自动转化成long类型的原理。
